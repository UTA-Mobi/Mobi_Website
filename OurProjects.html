Campus Synergy

May 2013 – Present Campus Synergy is an event sharing application. SAAMD sat in front of the whiteboard for months trying to revolutionize how events are publicized on campus. This application attempts to use software to provide a more interactive bridge to connect organizations with students. This app does not aim to put chalking, flyers, nor emails out of business. Campus Synergy will make event sharing more interactive and fun. The campus synergy (whiteboard days) design tried to revolutionize all student communication. With this new Campus Synergy, our aims are to concentrate on improving just the event sharing aspect of campus life.


UTA Radio App

November 2012 – March 2013

The broadcasting station at UTA wanted us to make an app for them. This would be a cross-platform web app that would stream audio over http and rtmp, integrate twitter, facebook, and allow users to request songs in realtime.

responsibilities: social media integration (twitter and facebook), meeting with UTA radio, creating an easy to use and smooth UI experience, and streaming audio over http and rtmp.

Frameworks: Phonegap, jQuery Mobile
Languages: HTML5, Javascript, and CSS

WebXCavate

August 2012 – August 2012 webXcavate is a caching game. The users of this app tag an eLocation with helpful hints, a question, and an answer. Users that want to play then would view a snapshot of the eLocation, read the hints, and surf the Web following the hints in order to name that eLocation.

Campus Synergy-First Time Around

Campus synergy is an interactive mobile application for college events. The application has a lot of features that would transform campuses into socially aware entities. There are a lot of common features between campus synergy and other official university apps yet campus synergy will change the way students interact with these apps; from an informative tool into an immersive experience. We will hold back on the details until the app is ready for prime time!
