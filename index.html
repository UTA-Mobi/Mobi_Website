Campus Synergy

Campus Synergy is an event sharing application. This application attempts to use software to provide a more interactive bridge to connect organizations with students. Campus Synergy will make event sharing more interactive and fun.

UTA Radio

The broadcasting station at UTA wanted us to make an app for them. This would be a cross-platform web app that would stream audio over http and rtmp, integrate twitter, facebook, and allow users to request songs in realtime.

WebXCavate

webXcavate is a caching game. The users of this app tag an eLocation with helpful hints, a question, and an answer. Users that want to play then would view a snapshot of the eLocation, read the hints, and surf the Web following the hints in order to name that eLocation.
